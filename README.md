# Exepnse Tracker
Javascript application for expenses tracking.

The purpose of the project was to learn JavaScript:
- classes
- modules import/export
- arrow function
- higher-order functions
- DOM manipulation
- events
- local storage
- chart creation

## Usage
Clone the repo and open index.html
